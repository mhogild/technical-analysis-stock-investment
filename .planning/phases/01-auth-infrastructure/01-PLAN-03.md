This is the security core of the Saxo integration.

**Constructor:**
```python
import asyncio
import secrets
from datetime import datetime, timedelta, timezone
from typing import Dict, Optional
from cryptography.fernet import Fernet
import httpx

from config import (
    SUPABASE_URL,
    SUPABASE_SERVICE_ROLE_KEY,
    SAXO_APP_KEY,
    SAXO_APP_SECRET,
    SAXO_REDIRECT_URI,
    SAXO_TOKEN_URL,
    SAXO_TOKEN_ENCRYPTION_KEY,
    SAXO_REFRESH_BUFFER_SECONDS,
    SAXO_CIRCUIT_BREAKER_LIMIT,
)
from models.saxo import SaxoTokenRecord
from services.saxo_exceptions import (
    SaxoNotConnectedError,
    SaxoCircuitBreakerOpenError,
    SaxoTokenExpiredError,
    SaxoOAuthError,
)


class SaxoTokenService:
    def __init__(self):
        self._fernet = Fernet(SAXO_TOKEN_ENCRYPTION_KEY.encode())
        self._refresh_locks: Dict[str, asyncio.Lock] = {}
        self._supabase_headers = {
            "apikey": SUPABASE_SERVICE_ROLE_KEY,
            "Authorization": f"Bearer {SUPABASE_SERVICE_ROLE_KEY}",
            "Content-Type": "application/json",
            "Prefer": "return=representation",
        }
```

**Methods to implement:**

1. `_encrypt(self, plaintext: str) -> str` — calls `self._fernet.encrypt(plaintext.encode()).decode()`

2. `_decrypt(self, ciphertext: str) -> str` — calls `self._fernet.decrypt(ciphertext.encode()).decode()`

3. `_get_lock(self, user_id: str) -> asyncio.Lock` — returns existing lock or creates new one for user_id. Implementation:
```python
    def _get_lock(self, user_id: str) -> asyncio.Lock:
        if user_id not in self._refresh_locks:
            self._refresh_locks[user_id] = asyncio.Lock()
        return self._refresh_locks[user_id]
```

4. `async def create_oauth_state(self, user_id: str) -> str` — generates `state = secrets.token_urlsafe(32)`, stores `(state, user_id, expires_at=now+10min)` in `saxo_oauth_state` table via POST to `{SUPABASE_URL}/rest/v1/saxo_oauth_state`. Returns the state string.

5. `async def validate_oauth_state(self, state: str) -> str` — fetches the state row from `saxo_oauth_state` via GET with `?state=eq.{state}`, verifies `expires_at > now`, extracts `user_id`, deletes the row, returns `user_id`. Raises `ValueError` if state not found or expired.

6. `async def exchange_code_for_tokens(self, code: str, user_id: str) -> None` — POST to `SAXO_TOKEN_URL` with `grant_type=authorization_code`, `code`, `redirect_uri=SAXO_REDIRECT_URI`, `client_id=SAXO_APP_KEY`, `client_secret=SAXO_APP_SECRET`. Content-Type must be `application/x-www-form-urlencoded`. Parse response JSON for `access_token`, `refresh_token`, `expires_in`, `refresh_token_expires_in`. Encrypt both tokens. Compute `expires_at = now + timedelta(seconds=expires_in)`. Upsert into `saxo_tokens` via POST with `Prefer: resolution=merge-duplicates` header or PATCH if row exists. If Saxo returns an error, parse `error` and `error_description` fields and raise `SaxoOAuthError(error, error_description)`.

7. `async def get_token_record(self, user_id: str) -> Optional[SaxoTokenRecord]` — fetches from `saxo_tokens` via GET `?user_id=eq.{user_id}`. Returns `SaxoTokenRecord` or `None` if no row.

8. `async def get_valid_token(self, user_id: str) -> str` — the core method:
```python
    async def get_valid_token(self, user_id: str) -> str:
        record = await self.get_token_record(user_id)
        if not record:
            raise SaxoNotConnectedError(user_id)

        if record.consecutive_refresh_failures >= SAXO_CIRCUIT_BREAKER_LIMIT:
            raise SaxoCircuitBreakerOpenError(user_id)

        now = datetime.now(timezone.utc)
        buffer = timedelta(seconds=SAXO_REFRESH_BUFFER_SECONDS)

        if record.expires_at - now < buffer:
            async with self._get_lock(user_id):
                # Re-fetch after acquiring lock (another coroutine may have refreshed)
                record = await self.get_token_record(user_id)
                if not record:
                    raise SaxoNotConnectedError(user_id)
                if record.consecutive_refresh_failures >= SAXO_CIRCUIT_BREAKER_LIMIT:
                    raise SaxoCircuitBreakerOpenError(user_id)
                if record.expires_at - now < buffer:
                    await self._refresh_and_store(user_id, record)
                    record = await self.get_token_record(user_id)

        return self._decrypt(record.access_token)
```

9. `async def _refresh_and_store(self, user_id: str, record: SaxoTokenRecord) -> None` — decrypts the refresh_token, POST to `SAXO_TOKEN_URL` with `grant_type=refresh_token`, `refresh_token`, `client_id=SAXO_APP_KEY`, `client_secret=SAXO_APP_SECRET`. On success: encrypt new tokens, update row in `saxo_tokens` with new `access_token`, `refresh_token`, `expires_at`, `refresh_expires_at`, `consecutive_refresh_failures=0`, `updated_at=now()`. On failure: increment `consecutive_refresh_failures` in the DB row via PATCH. If failures reach `SAXO_CIRCUIT_BREAKER_LIMIT`, raise `SaxoCircuitBreakerOpenError`.

10. `async def revoke_and_delete(self, user_id: str) -> None` — fetches token record, decrypts refresh_token, POST to `{SAXO_TOKEN_URL}/revoke` (note: the revoke URL is the token URL path + `/revoke`, which means for SIM it is `https://sim.logonvalidation.net/token/revoke`) with `token={refresh_token}&client_id={SAXO_APP_KEY}&client_secret={SAXO_APP_SECRET}` as form-encoded body. Best-effort: if revocation fails, log warning but continue. Then DELETE the `saxo_tokens` row via `DELETE {SUPABASE_URL}/rest/v1/saxo_tokens?user_id=eq.{user_id}`. Also clean up any lock for the user from `self._refresh_locks`.

**All Supabase HTTP calls** use `httpx.AsyncClient()` with `self._supabase_headers`. Use `async with httpx.AsyncClient() as client:` for each call, or accept an `httpx.AsyncClient` parameter. Raise on non-2xx Supabase responses with descriptive error messages.

**Security rule:** Never log the plaintext value of access_token or refresh_token. Log user_id and operation name only.
